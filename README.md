# 🎓 Education Analytics Dashboard – Power BI

This project is an interactive **Education Analytics Dashboard** developed using **Microsoft Power BI** to analyze and visualize key educational performance metrics. The dashboard provides meaningful insights into student performance, attendance, course enrollment, and overall academic trends through dynamic and user-friendly visualizations.

The project includes interactive charts, KPI cards, slicers, and filters that allow users to explore data from different perspectives. It helps identify high-performing students, monitor attendance patterns, compare subject-wise performance, and evaluate institutional performance effectively. The dashboard is designed to support data-driven decision-making for educators, administrators, and stakeholders.

The data preparation process involved cleaning, transforming, and modeling the dataset using **Power Query**, while **DAX (Data Analysis Expressions)** was used to create calculated measures and KPIs. A star schema data model was implemented to improve performance and ensure accurate analysis.

This project demonstrates my skills in **data cleaning, data modeling, DAX, Power Query, data visualization, and dashboard design**. It reflects my ability to transform raw educational data into clear, actionable insights using Power BI. This repository contains the Power BI project file, dataset (if applicable), and dashboard screenshots for reference. The dashboard is built with a focus on usability, performance, and effective storytelling through data visualization.
